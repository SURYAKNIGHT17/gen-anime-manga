Manga Creator API

Welcome to the Manga Creator API. This API allows you to generate manga stories and panels based on your prompts.

Endpoints

POST /api/story/generate

Generate a story based on a prompt and characters.

Request Body:

{ "prompt": "A hero's journey in a cyberpunk world", "characters": ["Cyber ninja", "AI companion"] }

POST /api/generate/panel

Generate a manga panel based on a scene description.

Request Body:

{ "scene": "A cyber ninja fighting robots in a neon-lit alley", "characters": ["Cyber ninja", "Robots"], "style": "manga" }

POST /api/export/pdf

Export manga as PDF.

Request Body:

{ "panels": [ { "path": "path/to/panel1.png", "dialogue": [ {"character": "Character 1", "text": "Dialogue text"} ] } ], "title": "My Manga" }

POST /api/export/cbz

Export manga as CBZ.

Request Body:

{ "panels": [ {"path": "path/to/panel1.png"} ], "title": "My Manga" }

For more information, check the GitHub repository.
